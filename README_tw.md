## 技術棧
- Bootstrap v5.2.3
- Google Maps JavaScript API
- Vue.js v3
- Vue Router
- Vue I18n
- Axios
## 需求
- Web 瀏覽器
## 功能
- 在地圖上標示監測站的位置
- 顯示監測站的空氣品質資料
- 可以更改語言（繁體中文、英文）
- 身分驗證
- 顯示使用者的設定內容
## 使用
- 在命令列工具中移動到專案的前端目錄
- 執行 `yarn` 安裝必要的套件
- 執行 `yarn dev`，啟動前端網站預覽模式
